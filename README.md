# COB (Catalyst Orchestrated Builder)

> [!WARNING]
> COB is still in very active beta-development, and features might
> change, and code might have bugs. If you encounter any in testing,
> please file an issue.

COB is a high-speed, low-level build orchestrator designed as a machine-
generated target for higher-level meta-build systems, focusing on performance
and dependency tracking.

Check out the [docs](https://catalystcpp.github.io/catalyst-orchestrated-builder/).
